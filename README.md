## 小程序前台 mpvue
基于 Vue.js 的小程序开发框架，从底层支持 Vue.js 语法和构建工具体系。

### 为什么要使用mpvue
``` bash
1.一套vue程序h5+小程序

2.Vue组件规范，熟悉vuejs即可开发小程序

3.比wepy工具对vue语法支持度更高

4.github上面星星很高，反应良好
```

### 小程序的缺点
``` bash
1.语法和html css不兼容，单独开发

2.不能用npm

3.学习成本高
```
